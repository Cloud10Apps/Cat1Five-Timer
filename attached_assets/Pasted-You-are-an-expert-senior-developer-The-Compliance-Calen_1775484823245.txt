THE FIX — Replace it entirely with this cleaner version.
The sql IS NOT NULL template tags are removed entirely. PostgreSQL's 
gte/lte operators already exclude NULL values implicitly — any comparison 
with NULL returns NULL (falsy) in PostgreSQL, so the NULL checks are 
redundant and cause type conflicts with Drizzle.

Also add isNotNull to the drizzle-orm import.
