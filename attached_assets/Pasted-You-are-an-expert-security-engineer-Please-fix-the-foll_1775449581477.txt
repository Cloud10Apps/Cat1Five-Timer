## 9. MEDIUM: No Subscription Enforcement on API Routes
FILE: artifacts/api-server/src/middleware/auth.ts (or a new middleware file)

PROBLEM: Users can sign up, skip billing, and use the full app forever with no payment. If this is a paid product, API routes are not enforcing active subscription status.

FIX: Create a requireActiveSubscription middleware:

export async function requireActiveSubscription(req: Request, res: Response, next: NextFunction) {
  const orgId = req.user!.organizationId;
  const [org] = await db.select().from(organizationsTable).where(eq(organizationsTable.id, orgId));
  
  // Allow if no Stripe is configured (dev mode)
  if (!process.env.STRIPE_SECRET_KEY) return next();
  
  // Allow if org has active or trialing subscription
  if (org.stripeSubscriptionId) {
    const { stripeStorage } = await import("../stripeStorage.js");
    const sub = await stripeStorage.getSubscription(org.stripeSubscriptionId);
    if (sub && (sub.status === "active" || sub.status === "trialing")) {
      return next();
    }
  }
  
  res.status(402).json({ error: "Active subscription required." });
}

Then apply this middleware to protected routes after requireAuth. Consider giving a grace period (e.g., 14 days from account creation) before enforcing.

## 10. MEDIUM: Stripe N+1 API Calls
FILE: artifacts/api-server/src/stripeStorage.ts

CURRENT CODE:
for (const product of products.data) {
  const prices = await stripe.prices.list({ product: product.id, ... });
}

PROBLEM: One Stripe API call per product — slow and inefficient.

FIX: Fetch all prices in one call with product expansion:
async getProductsWithPrices() {
  const stripe = await getUncachableStripeClient();
  const prices = await stripe.prices.list({
    active: true,
    limit: 100,
    expand: ["data.product"],
  });
  
  return prices.data
    .filter(price => price.recurring && typeof price.product !== "string" && (price.product as any).active)
    .map(price => {
      const product = price.product as Stripe.Product;
      return {
        product_id: product.id,
        product_name: product.name,
        product_description: product.description,
        price_id: price.id,
        unit_amount: price.unit_amount,
        currency: price.currency,
        recurring: price.recurring,
      };
    });
}
